# File and Directory Descriptions

This document provides an overview of the files and directories within the `benchmarking_llm_against_prompt_formats` environment.

*   `01_Manuscripts/`: Contains academic papers and manuscripts related to the experiments.
*   `02_RawData/`: This directory is intended to store the raw output logs from the benchmark runs.
    *   `lojban_test_results.json`: Raw JSON output from the `test_lojban_understanding.py` script.
    *   `lojban_advanced_test_results.json`: Raw JSON output from the `test_lojban_advanced.py` script.
*   `03_ExperimentalScripts/`: Contains all the scripts necessary to run and analyze the experiments.
    *   `parse_results.py`: A Python script to parse the results from all benchmark runs (natural language and symbolic) and generate a comprehensive summary report.
    *   `run_copy_list_test.py`: A Python script that runs the 'copy_list' code generation task.
    *   `run_custom_sort_test.py`: A Python script that runs the 'custom_sort' code generation task.
    *   `run_diagnosis_logic_test.py`: A Python script that runs the Diagnosis Logic puzzle. It supports S-expression (`s_expr`), JSON (`json`), TSV (`tsv`), and custom token-based (`token_test`) prompt formats.
    *   `run_einstein_riddle_test.py`: A Python script that runs the Einstein Riddle logic puzzle. It supports S-expression (`s_expr`), JSON (`json`), token-based (`token_test`), and Chain-of-Thought (`cot_baseline`) prompt formats.
    *   `run_experiments.sh`: The main orchestrator shell script that runs all benchmark suites (`simple_tasks`, `roundtrip`, `diagnosis`, `einstein`, `contradictory_test`). It can iterate through multiple models, prompt formats, languages, and prompt styles.

    *   `run_filtered_list_test.py`: A Python script that runs the 'Filtered List' reasoning and instruction-following task.
    *   `run_final_symbolic_benchmark.py`: A Python script to run the simple Fibonacci S-expression test.
    *   `run_length_sort_test.py`: A Python script that runs the 'length_sort' code generation task.
    *   `run_return_one_test.py`: A Python script that runs the 'return_one' code generation task.
    *   `run_reverse_sort_test.py`: A Python script that runs the 'reverse_sort' code generation task.
    *   `run_simple_sort_test.py`: A Python script that runs the 'simple_sort' code generation task.
    *   `calculate_overall_ranking.py`: A Python script that parses the aggregated markdown summary file and calculates the final overall success rate for each model, outputting a sorted markdown table.
    *   `test_lojban_understanding.py`: A supplementary Python script to run a basic Lojban-to-English translation test against all models.
    *   `test_lojban_advanced.py`: A supplementary Python script for a more comprehensive Lojban benchmark, including bidirectional translation (Lojban <=> en/ja/eo) and a simple code generation task.
*   `04_Prompts/`: Contains the prompt files and data used for the various experiments.
    *   `en/`, `ja/`, `eo/`, `jbo/`: Contain natural language prompts and prompt templates for all tests.
    *   `einstein.json`: The JSON data file defining the Einstein Riddle.

    *   `prompt_custom_sort_*.md`: Prompt files for the custom sort task in multiple languages.
    *   `prompt_filtered_list_*.txt`: Prompt files for the Filtered List task in multiple languages.
    *   `prompt_diagnosis_logic_*.md`: Prompt templates for the diagnosis test (json, s_expr, tsv, token_test).
    *   `prompt_einstein_riddle_*.md`: Prompt templates for the Einstein Riddle test (json, sxp, token_test).
    *   `ja/structured_code_generator.json`, `ja/code_to_structured_json.json`: Prompt templates for some round-trip code generation tests.
*   `06_TestDefinitions/`: Contains the static pytest test definitions used to validate the code generated by the LLMs.

    *   `test_copy_list.py`: Pytest definitions for the 'copy_list' task.
    *   `test_custom_sort.py`: Pytest definitions for the 'custom_sort' task.
    *   `test_diagnosis_logic.py`: Pytest definitions for the Diagnosis Logic puzzle.
    *   `test_einstein_riddle.py`: Pytest definitions for the Einstein Riddle puzzle.
    *   `test_length_sort.py`: Pytest definitions for the 'length_sort' task.
    *   `test_return_one_test.py`: Pytest definitions for the 'return_one' task.
    *   `test_reverse_sort.py`: Pytest definitions for the 'reverse_sort' task.
    *   `test_simple_sort.py`: Pytest definitions for the 'simple_sort' task.
*   `07_References/`: Contains reference materials, such as academic papers cited in the manuscripts.
*   `99_Archive/`: An archive for old or deprecated files.
*   `README.md`: The main documentation file for this experimental environment.
*   `requirements.txt`: Python package dependencies.
